# K230 通信协议说明

本文档对应当前工作区代码，更新时间为 `2026-07-14`。
对应文件：

- `main.py`
- `stand_aiming.py`
- `move_aiming.py`
- `circle_mode.py`

## 1. 串口参数

当前运行模式统一使用：

- UART：`UART2`
- TX：`GPIO5`
- RX：`GPIO6`
- 波特率：`115200`
- 文本编码：`UTF-8`
- 行结束：`\n`
- 数据格式：默认 `8N1`

## 2. 模式入口

当前 `main.py` 中 `FLAG` 映射如下：

| `FLAG` | 模式名 | 对应文件 |
|---|---|---|
| `0` | `calibrate` | `calibrate.py` |
| `1` | `idle` | 不启动任何模式 |
| `2` | `stand_aiming` | `stand_aiming.py` |
| `3` | `stand_aiming` | `stand_aiming.py` |
| `4` | `move_aiming` | `move_aiming.py` |
| `5` | `move_aiming` | `move_aiming.py` |
| `6` | `circle_mode` | `circle_mode.py` |

启动按键：

- `GPIO20`：`MODE_BTN`
- 启动时不按：直接按 `main.py` 顶部 `FLAG` 启动
- 启动时按住：进入临时 `FLAG` 单键轮换
- 每按一次，`FLAG` 按 `0 -> 1 -> 2 -> 3 -> 4 -> 5 -> 6 -> 0` 循环
- `FLAG=1` 时不启动任何模式

运行模式开始通信按键：

- `GPIO28`：`START_BTN`
- 仅在 `stand_aiming / move_aiming / circle_mode` 中生效
- 按下后才开始与下位机进行串口通信

## 3. stand_aiming 协议

`stand_aiming.py` 用于识别矩形框中心，并输出矩形框中心相对目标点的像素偏差。

说明：

- 目标点与屏幕中心在同一条竖直线中
- 目标点纵向偏移由 `TARGET_POINT_OFFSET_Z` 决定
- 程序启动后按下 `GPIO28` 前，不向下位机发送任何串口数据

### 3.1 K230 -> 下位机

未识别到有效矩形框时：

```text
RECT,SEARCH
```

识别到矩形框时：

```text
RECT,{dx},{dy},{aligned}
```

字段说明：

- `dx`：矩形框中心相对目标点的 X 偏差，单位为像素
- `dy`：矩形框中心相对目标点的 Y 偏差，单位为像素
- `aligned`：是否已进入对准范围
  - `0` 表示未对准
  - `1` 表示已对准

当首次进入对准范围时，还会额外发送：

```text
RECT_OK
```

### 3.2 状态消息

按下 `GPIO28` 开始通信后会发送：

```text
STS,RECT_CENTER_READY
```

## 4. move_aiming 协议

`move_aiming.py` 用于识别靶面、靶心和激光点，输出激光点相对靶心的物理偏差。

程序启动后按下 `GPIO28` 前，不向下位机发送任何串口数据。

### 4.1 K230 -> 下位机

未识别到有效目标时：

```text
AIM,SEARCH
```

识别到有效目标时：

```text
AIM,{dx_cm},{dy_cm},{distance_cm},{angle_deg}
```

字段说明：

- `dx_cm`：激光点相对靶心的 X 偏差，单位 `cm`
- `dy_cm`：激光点相对靶心的 Y 偏差，单位 `cm`
- `distance_cm`：激光点到靶心的距离，单位 `cm`
- `angle_deg`：激光点相对靶心的角度，单位 `deg`

示例：

```text
AIM,-1.35,0.82,1.58,148.7
```

### 4.2 状态消息

按下 `GPIO28` 开始通信后，模式切换到 `aim` 时会发送：

```text
STS,AIM
```

## 5. circle_mode 协议

`circle_mode.py` 用于圆环运动模式，发送激光点与圆环目标点之间的误差值。

程序启动后按下 `GPIO28` 前，不向下位机发送任何串口数据。

### 5.1 K230 -> 下位机

未识别到有效目标时：

```text
CIR,SEARCH
```

识别到有效目标时：

```text
CIR,{err_x_cm},{err_y_cm},{target_x_cm},{target_y_cm}
```

字段说明：

- `err_x_cm`：激光点相对目标点的 X 误差，单位 `cm`
- `err_y_cm`：激光点相对目标点的 Y 误差，单位 `cm`
- `target_x_cm`：当前圆环目标点的 X 坐标，单位 `cm`
- `target_y_cm`：当前圆环目标点的 Y 坐标，单位 `cm`

### 5.2 状态消息

按下 `GPIO28` 开始通信后会按状态发送：

```text
STS,IDLE
STS,WAITING
STS,RUNNING
STS,STOPPED
```
